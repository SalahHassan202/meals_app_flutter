
# Flutter Meals App

A simple and clean Meals App built with **Flutter**.  
The app includes an onboarding screen, a dynamic home page, and a meals list.  
Users can add new meals and view full meal details.  
The project is great for learning UI design, navigation, and local data storage using **SqfLite** and **Shared Preferences**.

Project Overview

This project shows how to build a small meals application with a clean structure.
It is great for beginners who want to learn:

How to organize screens and widgets

How to work with lists and dynamic content

How to store data locally using SqfLite

How to save small values with Shared Preferences

How to design simple and clear UI components
